## Deployment Instructions

<div id="deploy"></div>

<h3 id="deploy-vllm-agg-b200">vLLM agg B200</h3>

DP=4 + expert parallel path for B200.

```bash
export NAMESPACE=your-namespace

# Prepare model cache and model assets using setup manifests in recipes/deepseek-v4/deepseek-v4-flash.
kubectl apply -f recipes/deepseek-v4/deepseek-v4-flash/vllm/agg_b200/deploy.yaml -n ${NAMESPACE}
```
<h3 id="deploy-vllm-agg-gb200">vLLM agg GB200</h3>

TP=4 + expert parallel path for one GB200 NVL4 tray.

```bash
export NAMESPACE=your-namespace

# Prepare model cache and model assets using setup manifests in recipes/deepseek-v4/deepseek-v4-flash.
kubectl apply -f recipes/deepseek-v4/deepseek-v4-flash/vllm/agg_gb200/deploy.yaml -n ${NAMESPACE}
```
<h3 id="deploy-sglang-agg-b200">SGLang agg B200</h3>

SGLang TP4 with MXFP4 MoE and EAGLE MTP.

```bash

# Prepare model cache and model assets using setup manifests in recipes/deepseek-v4/deepseek-v4-flash.
```
