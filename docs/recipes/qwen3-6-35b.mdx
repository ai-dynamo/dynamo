## Deploy

<h3 id="deploy-vanilla-vllm-serve">Deploy: vanilla vLLM serve</h3>

Plain vLLM baseline outside DynamoGraphDeployment.

```bash
export NAMESPACE=your-namespace
export HW=gb200  # or h100

# Prepare shared storage and model assets using the setup manifests in `recipes/qwen3.6-35b`.
kubectl apply -f recipes/qwen3.6-35b/deploy/vllm-serve.yaml -n ${NAMESPACE}
# Or run this recipe's script-driven benchmark for the selected variant.
./run-benchmark.sh -n ${NAMESPACE} --hw ${HW} --config vllm-serve
```

<h3 id="deploy-dynamo-frontend-decoding">Deploy: Dynamo frontend decoding</h3>

Tests Dynamo frontend decoding with multimodal processing on and embedding cache off.

```bash
export NAMESPACE=your-namespace
export HW=gb200  # or h100

# Prepare shared storage and model assets using the setup manifests in `recipes/qwen3.6-35b`.
kubectl apply -f recipes/qwen3.6-35b/deploy/dynamo-fd.yaml -n ${NAMESPACE}
# Or run this recipe's script-driven benchmark for the selected variant.
./run-benchmark.sh -n ${NAMESPACE} --hw ${HW} --config dynamo-fd
```

<h3 id="deploy-dynamo-fd--embedding-cache">Deploy: Dynamo FD + embedding cache</h3>

Tests frontend decoding plus 8 GiB multimodal embedding cache.

```bash
export NAMESPACE=your-namespace
export HW=gb200  # or h100

# Prepare shared storage and model assets using the setup manifests in `recipes/qwen3.6-35b`.

```
