## Deploy

<h3 id="deploy-vllm-aggregated">Deploy: vLLM aggregated</h3>

Single-node TP4 baseline and smallest hardware footprint.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/llama-3-70b`.
kubectl apply -f recipes/llama-3-70b/vllm/agg/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/llama-3-70b/vllm/agg/perf.yaml -n ${NAMESPACE}
```

<h3 id="deploy-vllm-disagg-single-node">Deploy: vLLM disagg single-node</h3>

P/D split within one node; compare latency against aggregate baseline.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/llama-3-70b`.
kubectl apply -f recipes/llama-3-70b/vllm/disagg-single-node/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/llama-3-70b/vllm/disagg-single-node/perf.yaml -n ${NAMESPACE}
```
