## Deployment Instructions

<div id="deploy"></div>

<h3 id="deploy-vllm-agg-b200">vLLM agg B200</h3>

Single-node TP=8 + expert-parallel path for B200.

```bash
export NAMESPACE=your-namespace

# Prepare model cache and model assets using setup manifests in recipes/deepseek-v4/deepseek-v4-pro.
kubectl apply -f recipes/deepseek-v4/deepseek-v4-pro/vllm/agg/b200/deploy.yaml -n ${NAMESPACE}
```
<h3 id="deploy-vllm-agg-gb200">vLLM agg GB200</h3>

## Smoke Test

<div id="smoke-test"></div>

Run a small request before starting the benchmark. Replace the service name if your deployment exposes a different frontend.

```bash
export NAMESPACE=your-namespace
kubectl port-forward svc/<frontend-service> 8000:8000 -n ${NAMESPACE}

curl http://localhost:8000/v1/chat/completions \
  -H 'Content-Type: application/json' \
  -d '{"model":"deepseek-ai/DeepSeek-V4-Pro","messages":[{"role":"user","content":"Write a one-sentence readiness check."}],"max_tokens":64}'
```
