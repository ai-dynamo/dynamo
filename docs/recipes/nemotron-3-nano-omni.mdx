## Deployment Instructions

Use for functional multimodal deployment; add perf.yaml before making performance claims.

```bash
export NAMESPACE=your-namespace

# Prepare model cache and model assets using setup manifests in recipes/nemotron-3-nano-omni.
kubectl apply -f recipes/nemotron-3-nano-omni/vllm/agg/deploy.yaml -n ${NAMESPACE}
```
