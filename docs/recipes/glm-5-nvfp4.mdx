## Deploy

<h3 id="deploy-sglang-disaggregated-prefill-decode">Deploy: SGLang disaggregated prefill/decode</h3>

Use as the current benchmark-backed GLM-5 path; it stresses long output with a large decode rank group.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/glm-5-nvfp4`.
kubectl apply -f recipes/glm-5-nvfp4/sglang/disagg/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/glm-5-nvfp4/sglang/disagg/perf.yaml -n ${NAMESPACE}
```
