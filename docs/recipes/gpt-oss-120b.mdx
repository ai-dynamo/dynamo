## Deploy

<h3 id="deploy-tensorrt-llm-aggregated">Deploy: TensorRT-LLM aggregated</h3>

Use for short-prompt, long-output, high-concurrency throughput testing.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/gpt-oss-120b`.
kubectl apply -f recipes/gpt-oss-120b/trtllm/agg/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/gpt-oss-120b/trtllm/agg/perf.yaml -n ${NAMESPACE}
```

<h3 id="deploy-tensorrt-llm-disaggregated">Deploy: TensorRT-LLM disaggregated</h3>

Use for long-context generation where P/D separation is the target shape.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/gpt-oss-120b`.
kubectl apply -f recipes/gpt-oss-120b/trtllm/disagg/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/gpt-oss-120b/trtllm/disagg/perf.yaml -n ${NAMESPACE}
```
