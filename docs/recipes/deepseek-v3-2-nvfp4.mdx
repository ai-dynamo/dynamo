## Deploy

<h3 id="deploy-disagg--kv-router--wideep">Deploy: Disagg + KV router + WideEP</h3>

Tests KV-aware routing, P/D split, and WideEP together on the target trace.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/deepseek-v32-fp4`.
kubectl apply -f recipes/deepseek-v32-fp4/trtllm/disagg-kv-router/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/deepseek-v32-fp4/trtllm/disagg-kv-router/perf.yaml -n ${NAMESPACE}
```

<h3 id="deploy-aggregated-round-robin">Deploy: Aggregated round-robin</h3>

Keeps DEP8 and hardware constant while removing the P/D split and KV-aware routing.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/deepseek-v32-fp4`.
kubectl apply -f recipes/deepseek-v32-fp4/trtllm/agg-round-robin/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/deepseek-v32-fp4/trtllm/agg-round-robin/perf.yaml -n ${NAMESPACE}
```
