## Deploy

<h3 id="deploy-aggregated-hopper">Deploy: Aggregated Hopper</h3>

Use on Hopper; default MoE backend.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/qwen3-235b-a22b-fp8`.
kubectl apply -f recipes/qwen3-235b-a22b-fp8/trtllm/agg/hopper/deploy.yaml -n ${NAMESPACE}
# Run the benchmark for this variant.
kubectl apply -f recipes/qwen3-235b-a22b-fp8/trtllm/agg/hopper/perf.yaml -n ${NAMESPACE}
```
