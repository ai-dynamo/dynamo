## Deploy

<h3 id="deploy-vllm-aggregated">Deploy: vLLM aggregated</h3>

TP4 aggregate path with approximate KV-aware routing and Mamba cache workaround.

```bash
export NAMESPACE=your-namespace


# Prepare shared storage and model assets using the setup manifests in `recipes/nemotron-3-super-fp8`.
kubectl apply -f recipes/nemotron-3-super-fp8/vllm/agg/deploy.yaml -n ${NAMESPACE}
# Benchmark placeholder: no perf.yaml is checked in for this variant yet.
```
