## Recipes And Benchmarks

- Recipes are runnable targets for a model, hardware target, runtime, and workload.
- Benchmarks compare a feature, topology, or feature stack under controlled traffic.
- When a benchmark identifies a deployment target users should run, that target appears in Recipes and links back here.
