NVIDIA Dynamo is developed in the open. This page collects the places to ask questions, report
issues, and contribute, plus how to join the weekly community meeting.

## Get Involved

- **Source and issues** — file bugs and feature requests on the
  [ai-dynamo/dynamo](https://github.com/ai-dynamo/dynamo) GitHub repository.
- **Discussions** — ask questions and share ideas in
  [GitHub Discussions](https://github.com/ai-dynamo/dynamo/discussions).
- **CNCF Slack** — [join CNCF Slack](https://communityinviter.com/apps/cloud-native/cncf) and find
  us in the `#ai-dynamo` channel.
- **Contribute** — see the [Contribution Guide](../contribution-guide.md) to set up a development
  environment and open your first pull request.

## Community Meeting

The Dynamo community meets **weekly on Wednesdays at 10:30 AM Pacific (PST/PDT)**.

- **Join the call** — [Google Meet](https://meet.google.com/heb-demu-qok).
- **Meeting notes** — [agenda and running notes](https://docs.google.com/document/d/1uR8xD_hlYGwV6QspvSc36k1H-wo1BUcVmFbHH9xlXd8/edit)
  (add topics here before the meeting).

### Add to Your Calendar

Download the meeting invite and import it into your calendar to get the weekly event with the Google
Meet link:

- **[Download the calendar invite (.ics)](https://github.com/ai-dynamo/dynamo/blob/main/docs/assets/dynamo-community-meeting.ics)**

To import the `.ics` file:

- **Google Calendar** — open **Settings → Import & export → Import**, select the file, and choose a
  calendar.
- **Outlook** — open **File → Open & Export → Import/Export → Import an iCalendar (.ics) file**.
- **Apple Calendar** — open **File → Import** and select the file.
