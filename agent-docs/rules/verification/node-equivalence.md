# Node Equivalence

Record the serving nodes, GPU SKU and count, and MIG or time-slicing mode for the configuration under evaluation and
its baseline or prior reference. Prefer the same node pool. If the hardware or sharing mode differs, treat the
comparison as invalid rather than attributing the difference to the configuration.
